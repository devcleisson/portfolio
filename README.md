# Portfólio - José Cleisson

Site pessoal feito com HTML, CSS e JavaScript puro (sem framework, sem build).

## Como testar localmente

Só abrir o arquivo `index.html` no navegador. Não precisa instalar nada.

## Como publicar na Vercel

1. Crie um repositório novo no seu GitHub e suba estes 3 arquivos (`index.html`, `style.css`, `script.js`).
2. Entre em https://vercel.com e faça login com sua conta do GitHub.
3. Clique em "Add New Project" e selecione o repositório que você acabou de criar.
4. Não precisa mexer em nenhuma configuração de build (é um site estático). Clique em "Deploy".
5. Pronto — em menos de 1 minuto a Vercel te dá um link tipo `seu-projeto.vercel.app`.
